# CitizenPrep Landing Page

Static ASO/SEO landing page for `US Citizenship Test 2026`.

## Files

- `index.html`: landing page, SEO meta tags, Open Graph, Twitter card, app schema, FAQ schema.
- `styles.css`: responsive page styles.
- `assets/citizenprep-logo.png`: full CitizenPrep logo.
- `assets/citizenprep-app-icon.png`: app icon.

## Before Publishing

- Replace `https://example.com/` canonical URL with the production domain.
- Replace Google Play and App Store placeholder links after store approval.
- Add privacy policy URL link when available.
- Keep independent-app disclaimer visible.
